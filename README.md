# Embedded Systems Foundations

This repository documents my path from C programming fundamentals to Embedded Systems development. 
The goal is to understand how hardware and software interact at a deeper level.

---

## Goals

- Build strong Embedded Systems fundamentals
- Strengthen C for Embedded programming
- Understand microcontrollers and architecture
- Learn memory organization and registers
- Work with STM32 development tools
- Build hands-on projects
- Develop problem-solving and debugging skills
